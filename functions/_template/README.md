# Function Template

Copiar esta carpeta para crear una nueva Appwrite Function.

Renombrar:

- /functions/\_template -> /functions/<functionName>

Luego actualizar:

- README.md (proposito, trigger, scopes, env vars)
- .env.example
- src/main.js (implementacion)

Este template sigue documentation/04_appwrite_functions_rules.md.
